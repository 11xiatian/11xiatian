# 异或运算解析

## 文字理解

异或运算是位运算的一种，符号为：^
**相同为0，不同为1。** 容易与同或运算记混，根据异或的特性也叫 **不进位相加。**

展示例子

~~~ python

temp = 5^6    #异或运算
print(temp)   #这里temp输出的值为 3 
#原因：两个数字的二进制表示为     5----101
#                            6----110
#二进制相加=                        011
#  011  十进制表示为 3
~~~

上述  **5----101**

​          **6----110**

异或          **011**

**5**的第一位与**6**的第一位相同，所以得出的二进制第一位为**0**，后面两位都不同，所以得出的二进制都为**1**

即      **011----3**

对于二进制数比较长的直接二进制右对齐再**不进位相加**即可

## 多试几组数字自然就会理解异或运算了，之后可以做多个异或运算,就像乘法运算一样的
